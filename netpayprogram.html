Net Pay Program

Simple Net pay calculator. It takes your gross pay and then deducts Federal, State, FICA. It has the option to deduct any addtional amonut from your check. Then it displays your net pay.

  1. START
  2. DISPLAY "What is Gross Pay per check"
  3. Input GrossPay
  4. Set fed = .15
  5. SET state =.05
  6. SET fica = 0.762
  7. SET deductions
  8. DISPLAY "Are there any additional deductions"
  9. Input ans
  10. Process if ans === true/yes
    1. Display "Enter additinal amount:"
    2. Input additionalDeduction
    3. SET deductions = fed + state + fica + deductions
    4. SET netPay = grossPay - deductions

  12. END
